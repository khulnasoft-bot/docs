<RequestExample>

```bash cURL
curl --request POST \
--url https://api.teleflow.khulnasoft.com/v1/feeds \
--header 'Authorization: ApiKey <TELEFLOW_API_KEY>' \
--header 'Content-Type: application/json' \
--data '{"name": "<FEED_NAME>"}'
```

```javascript Node.js
import { Teleflow } from '@teleflow/node';

const teleflow = new Teleflow('<TELEFLOW_API_KEY>');
await teleflow.feeds.create("<Feed_Name>");
```

```php PHP
use Teleflow\SDK\Teleflow;

$teleflow = new Teleflow(<TELEFLOW_API_KEY>);

$teleflow->createFeed([
    'name' => '<insert-name-for-feed>'
]);
```

```ruby Ruby
require 'teleflow'

client = Teleflow::Client.new('<TELEFLOW_API_KEY>')

client.create_feed({
    'name' => 'New feed'
})
```

```python Python
from teleflow.api import FeedApi

url = "https://api.teleflow.khulnasoft.com"
api_key = "<TELEFLOW_API_KEY>"
teleflow = FeedApi(url, api_key).create(
    name = "<Feed_Name>",
)
```

