<RequestExample>
```javascript Node.js
import { Teleflow } from '@teleflow/node';

const teleflow = new Teleflow('<TELEFLOW_API_KEY>');

await teleflow.layouts.delete("layoutId");
```

```ruby Ruby
require 'teleflow'

client = Teleflow::Client.new('<TELEFLOW_API_KEY>')
client.delete_layout('<insert-layout-id>')
```

```python Python
from teleflow.api import LayoutApi

url = "https://api.teleflow.khulnasoft.com"
api_key = "<TELEFLOW_API_KEY>"
teleflow = LayoutApi(url, api_key).delete(
    layout_id="<Layout_ID to identify the layout to be deleted>",
)
```

```java Java
import co.teleflow.sdk.Teleflow;

public class Main {
    public static void main(String[] args) {
        String apiKey = "<TELEFLOW_API_KEY>";

        Teleflow teleflow = new Teleflow(apiKey);
        teleflow.deleteLayout(layoutId);
    }
}
```
