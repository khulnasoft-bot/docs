<RequestExample>
```javascript Node.js
import { Teleflow } from '@teleflow/node';

const teleflow = new Teleflow('<TELEFLOW_API_KEY>');

await teleflow.layouts.setDefault("layoutId");
```

```ruby Ruby
require 'teleflow'

client = Teleflow::Client.new('<TELEFLOW_API_KEY>')
client.make_default_layout('<insert-layout-id>')
```

```python Python
from teleflow.api import LayoutApi

url = "https://api.teleflow.khulnasoft.com"
api_key = "<TELEFLOW_API_KEY>"
teleflow = LayoutApi(url, api_key).set_default(
    layout_id= "<The layout_id corresponding to the layout to be set default>",
)
```

```java Java
import co.teleflow.sdk.Teleflow;

public class Main {
    public static void main(String[] args) {
        String apiKey = "<TELEFLOW_API_KEY>";

        Teleflow teleflow = new Teleflow(apiKey);
        teleflow.makeDefaultLayout(layoutId);
    }
}
```

</RequestExample>

{
