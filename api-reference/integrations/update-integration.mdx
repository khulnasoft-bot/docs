<RequestExample>

```javascript Node.js
import { Teleflow, ProvidersIdEnum } from '@teleflow/node'; 

const teleflow = new Teleflow('<TELEFLOW_API_KEY>');

const updatePayload = {
  name: "SendGrid",
  identifier: "sendgrid-identifier",
  credentials: {
    apiKey: "SUPER_SECRET_API_KEY",
    from: "no-reply@teleflow.khulnasoft.com",
    senderName: "Teleflow Team"
    // ... other credentials as per provider
  },
  active: true,
  check: false
}

await teleflow.integrations.getWebhookProviderStatus(ProvidersIdEnum.SendGrid, updatePayload);
```

```php PHP
// Get integrations
use Teleflow\SDK\Teleflow;

$teleflow = new Teleflow(<TELEFLOW_API_KEY>);
$teleflow->updateIntegration($integrationId, [
    'active' => true,
    'credentials' => [
        // insert all the fields
    ],
    'check' => true
])->toArray();
```

```ruby Ruby
require 'teleflow'

client = Teleflow::Client.new('<TELEFLOW_API_KEY>')
body = {
    'active' => true,
    'check' => true
    'credentials' => {
        # insert all the fields
    },
}
client.update_integration('<insert-provider-id>', body)
```

```python Python
from teleflow.api import IntegrationApi

url = "https://api.teleflow.khulnasoft.com"
api_key = "<TELEFLOW_API_KEY>"
teleflow = IntegrationApi(url, api_key).update(
    integration="<The instance of the integration to update>",
)
```
