Teleflow's PHP SDK provides simple, yet comprehensive notification management, and delivery capabilities through multiple channels that you can implement using code that integrates seamlessly with your PHP application.

[Explore the source code on GitHub](https://github.com/khulnasoft/teleflow-php)

## Installation

```php
composer require unicodeveloper/teleflow
```

<Tip> If you're ready to start integrating in your PHP app, jump straight to our [PHP quickstart.](/quickstarts/php)</Tip>

## Usage 

```php
use Teleflow\SDK\Teleflow;

$teleflow = new Teleflow(<INSERT_API_KEY_HERE>);

// Sign up on https://web.teleflow.khulnasoft.com and grab your API key from https://web.teleflow.khulnasoft.com/settings

$response = $teleflow->triggerEvent([
    'name' => '<REPLACE_WITH_TEMPLATE_NAME_FROM_ADMIN_PANEL>',
    'payload' => ['customVariables' => 'Hello'],
    'to' => [
        'subscriberId' => '<SUBSCRIBER_IDENTIFIER_FROM_ADMIN_PANEL>',
        'phone' => '07983882186'
    ]
])->toArray();
```
