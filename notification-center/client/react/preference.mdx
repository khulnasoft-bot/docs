By default, Teleflow will show the subscriber preferences cog icon on the notification center component. If you want to hide it, you can use the prop `showUserPreferences` on the `PopoverNotificationCenter` component.

Users can enable/disable any active channel in the workflow using subscriber preferences

<img
  style={{ borderRadius: "0.5rem" }}
  src="/images/manage-subscribers/user-preference.png"
/>

## Get subscriber preferences

<CodeGroup>

```typescript Node.JS
import { Teleflow } from "@teleflow/node";

const teleflow = new Teleflow("<TELEFLOW_API_KEY>");

// 111 is subscriberId
await teleflow.subscribers.getPreference("111");
```

</CodeGroup>

## Update subscriber preference for a workflow

<CodeGroup>

```typescript Node.JS
import { Teleflow, ChannelTypeEnum } from "@teleflow/node";

const teleflow = new Teleflow("<TELEFLOW_API_KEY>");

// enable in_app channel
await teleflow.subscribers.updatePreference("subscriberId", "workflowIdentifier", {
  enabled: true,
  channel: { type: ChannelTypeEnum.IN_APP, enabled: true },
});

// enable email channel
await teleflow.subscribers.updatePreference("subscriberId", "workflowIdentifier", {
  enabled: true,
  channel: { type: ChannelTypeEnum.EMAIL, enabled: true },
});
```

</CodeGroup>
